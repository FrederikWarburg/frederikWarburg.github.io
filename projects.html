Place Recognition (Master thesis)

Most of the content was published as a CVPR 2020 paper (see paper here).

We contribute with Mapillary Street-Level Sequences (MSLS), a large dataset for place recognition from image sequences. It contains more than 1.6 million images curated from the Mapillary collaborative mapping platform. The dataset is orders of magnitude larger than current data sources, and reflects the diversities of true lifelong learning. It features images from 30 major cities across six continents, hundreds of distinct cameras, and substantially different viewpoints and capture times, spanning all seasons over a nine-year period. All images are geo-located with GPS and compass, and feature high-level attributes such as road type. We propose a set of benchmark tasks designed to push state-of-the-art performance and provide baseline studies. We show that current state-of-the-art methods still have a long way to go, and that the lack of diversity in existing datasets has prevented generalization to new environments.

Burst Image Deblurring (re-implementation)

PyTorch implementation of "Burst Image Deblurring Using Permutation Invariant Convolutional Neural Networks".

The network takes a sequence of n burst images and outputs a single sharper image. Each burst image is fed through a siamese U-Net with skip connections. After each convolutional block, a global max pooling is applied to aggregate information across the images in the burst sequence. The feature maps are then collapsed into a cleaner output image.

TI-Pooling (re-implementation)

PyTorch implementation of TI-pooling (transformation-invariant pooling) from "TI-pooling: transformation-invariant pooling for feature learning in Convolutional Neural Networks".

TI-pooling is a simple technique that makes a Convolutional Neural Network (CNN) transformation-invariant. Given a set of nuisance transformations (such as rotations, scale, shifts, illumination changes, etc.), TI-pooling guarantees that the network's output does not depend on whether the input image was transformed.

Object Detection on Unsupervised Depth Estimates

This project estimates the 3D position of railroad furniture. We (1) adapt and train a 2D object detection model on the full Railroad dataset, and (2) estimate a dense depth map so COWI can recover the 3D positions of the detected objects from sequences of images.

AWS DeepRacer

The AWS DeepRacer was developed for research in deep reinforcement learning. However, performance with the built-in models and variants is limited. To improve trajectory tracking with predefined markers, we explore the viability of using other controllers.

We implemented a proportional controller and MPC on the DeepRacer car. Due to the limited time for this course project, we have implemented a proportional controller to the on-board computer and simulated MPC for tracking a spline trajectory.

Gender Bias in Neural Pronoun Resolution Models

Pronoun resolution is the task of linking a pronoun to the correct noun. We find that pre-trained state-of-the-art neural models tend to be biased. We show that one of the most commonly used datasets for training coreference resolution models has a substantial bias towards male entities, which causes models to perform better on male examples. Since this can lead to discrimination against, e.g., job applicants, we are motivated to highlight the problem and explore methods to mitigate this gender bias. We find that gender-neutral word embeddings, such as the Debiased GoogleNews embeddings or the Word Dependency embeddings, can considerably lower the bias in the model's predictions.

EKF SLAM for Eyewear (BSc. Thesis)

The objective of this thesis is to adapt and evaluate how well an open-source SLAM implementation works with data recorded from the Tobii Pro Glasses 2. The report investigates all aspects of the Extended Kalman Filter (EKF) SLAM method for data recorded with the Tobii Pro Glasses 2.
